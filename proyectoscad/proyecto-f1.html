Proyecto Mecedora Eliptica

Diseñar una pieza 3D utilizando CATIA y, posteriormente, emplear la herramienta Machining Process para simular su proceso de maquinado.

La práctica comenzó con la creación del modelo 3D de la pieza en el entorno Part Design de CATIA, donde se emplearon diversas herramientas de diseño como extrusiones, cortes y redondeos para obtener la geometría final deseada. Una vez diseñada la pieza, se procedió a preparar el entorno de mecanizado utilizando el módulo Machining.

En este módulo, se definieron parámetros esenciales como el tipo de material de la pieza, las herramientas de corte que se utilizarían, así como las trayectorias de mecanizado que simularían el proceso real de corte y desbaste del material.

Finalmente, se realizó la simulación del proceso de mecanizado para observar cómo la herramienta de corte eliminaba material y confirmaba que el proceso estaba bien definido. Con base en la simulación, se pudieron hacer ajustes a las trayectorias o parámetros del mecanizado para asegurar un resultado óptimo y eficiente.
